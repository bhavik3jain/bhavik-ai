# Investment Metrics Reference

## Formulas

### Cap Rate
```
Cap Rate = NOI / Purchase Price × 100
NOI = Gross Rent − Vacancy − Operating Expenses
```
Good: ≥6–8% | Red flag: <5%

### Cash-on-Cash (CoC)
```
CoC = Annual Cash Flow / Total Cash Invested × 100
Annual Cash Flow = NOI − Annual Debt Service
Total Cash Invested = Down Payment + Closing Costs + Rehab
```
Good: ≥6–10% | Minimum viable: 5%

### Gross Rent Multiplier (GRM)
```
GRM = Purchase Price / Gross Annual Rent
```
Good: <10 | Caution: >12

### DSCR
```
DSCR = NOI / Annual Debt Service
```
Lender min: 1.20–1.25x | Good: ≥1.30x | Below 1.0 = doesn't cover mortgage

### Monthly Mortgage Payment
```
M = P × [r(1+r)^n] / [(1+r)^n − 1]
P = loan amount | r = annual rate/12 | n = years × 12
```

## Expense Estimation (when not provided)

**50% Rule (quick screen):** NOI ≈ Gross Rent × 50%

| Expense | Typical % of Gross Rent |
|---|---|
| Property taxes | 8–12% |
| Insurance | 3–5% |
| Maintenance | 5–10% |
| Property management | 8–10% |
| Vacancy | 5–8% |
| CapEx reserve | 5–10% |
| **Total** | **~40–55%** |
