# Scoring Rubric

Score each listing across 6 dimensions. Final score = weighted average.

## Dimensions

### 1. Financial Returns (30%)
Compare cap rate and CoC against profile thresholds.
| Score | Criteria |
|---|---|
| 9–10 | Both cap rate AND CoC exceed targets by 2%+ |
| 7–8 | Both metrics meet or slightly exceed targets |
| 5–6 | One meets target, other is close |
| 3–4 | One or both fall short |
| 1–2 | Neither close to targets |

### 2. Location (25%)
Proximity to employment, schools, transit; neighborhood trajectory.
| Score | Criteria |
|---|---|
| 9–10 | A-class, strong demand, appreciating |
| 7–8 | B-class, solid demand, stable/improving |
| 5–6 | B-/C+, adequate demand, flat |
| 3–4 | C-class, softer demand, some risk |
| 1–2 | D-class, high vacancy risk, declining |

### 3. Property Condition (20%)
Infer from photos, year built, listing description, disclosed updates.
| Score | Criteria |
|---|---|
| 9–10 | Turnkey, recently renovated, no deferred maintenance |
| 7–8 | Good condition, minor cosmetic items only |
| 5–6 | Fair, some updates needed (~$10–25k) |
| 3–4 | Significant work needed ($25–75k) |
| 1–2 | Major rehab ($75k+), structural concerns |

### 4. Deal Structure (15%)
Price vs assessed, days on market, motivated seller signals.
| Score | Criteria |
|---|---|
| 9–10 | Below assessed value, long DOM, motivated seller |
| 7–8 | At/near assessed, some room to negotiate |
| 5–6 | Slight premium, typical market pricing |
| 3–4 | Well above assessed, limited flexibility |
| 1–2 | Significantly overpriced |

### 5. Risk Factors (10%)
HOA, flood zone, pre-1978 lead paint, short lease terms, age of major systems.
| Score | Criteria |
|---|---|
| 9–10 | No material risks |
| 7–8 | 1 minor risk factor |
| 5–6 | 2 minor or 1 moderate |
| 3–4 | Multiple moderate risks |
| 1–2 | Major risks (flood, structural, legal) |

### 6. Profile Fit (adjustment)
- +0.5 if property type matches profile preference
- +0.5 if location matches target markets
- −1.0 if any hard constraint from profile is violated

## Score Interpretation
| Score | Label | Action |
|---|---|---|
| 8.0–10.0 | Strong Buy | Prioritize — schedule showing immediately |
| 6.5–7.9 | Buy | Worth pursuing — run deeper diligence |
| 5.0–6.4 | Neutral | Consider only if price is negotiated down |
| 3.0–4.9 | Pass | Doesn't meet investment criteria |
| 1.0–2.9 | Hard Pass | Significant issues — avoid |
